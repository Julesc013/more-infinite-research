# More Infinite Research 0.6.0

Minimal finite museum edition for Factorio 0.6.4. It emits configurable numbered continuations for inserter capacity, rocket shooting speed, and rocket damage using target-era science packs.

Extract the release ZIP into the mods directory; Factorio 0.6.4 does not discover ZIP mods. Edit `config.lua` to disable a family or lower its finite bound.

Build with `./scripts/Build-MIRPackage.ps1`. The four-file archive contains no modern compiler/runtime, scripts, settings, infinite fields, or productivity research.
