# More Infinite Research 0.9.0

Finite museum edition for Factorio 0.9.8. It emits configurable numbered continuations for inserter capacity, gun-turret damage, bullet shooting speed, bullet damage, and toolbelt, using the four target-era science packs.

Edit `config.lua` in an extracted directory to disable families or lower their compiled bounds. Factorio 0.9.8 also discovers the deterministic release ZIP directly.

Build with `./scripts/Build-MIRPackage.ps1`. The ZIP contains only `info.json`, `config.lua`, `data.lua`, and the English locale. No native infinite fields, modern settings, productivity research, runtime compiler, or scripts ship.
