# More Infinite Research 0.8.0

Finite museum edition for Factorio 0.8.8. It emits configurable numbered continuations for inserter capacity, gun-turret damage, bullet shooting speed, bullet damage, and toolbelt with target-era science packs.

Factorio 0.8.8 does not discover ZIP mods. Extract the release ZIP into the mods directory, then edit `config.lua` if you want to disable a family or lower its compiled bound.

Build with `./scripts/Build-MIRPackage.ps1`. The archive contains only `info.json`, `config.lua`, `data.lua`, and English locale data. It contains no modern compiler/runtime, scripts, settings, native infinite fields, or productivity research.
