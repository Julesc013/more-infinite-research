# More Infinite Research 0.7.0

Finite museum edition for Factorio 0.7.5. It emits configurable numbered continuations for inserter capacity, gun-turret damage, bullet shooting speed, bullet damage, and toolbelt using target-era science packs.

Extract the release ZIP into the mods directory; Factorio 0.7.5 does not discover ZIP mods. Edit `config.lua` to disable a family or lower its finite bound.

Build with `./scripts/Build-MIRPackage.ps1`. The four-file archive ships no modern compiler/runtime, scripts, settings, infinite fields, or productivity research.
