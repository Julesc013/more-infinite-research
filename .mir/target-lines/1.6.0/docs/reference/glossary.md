# Glossary

| Term | Meaning |
| --- | --- |
| Capability | A resolver lane that classifies facts and may propose or diagnose behavior. |
| Claim | A public compatibility statement backed by evidence. |
| DecisionRecord | A diagnostic row explaining generated, skipped, rejected, or observed behavior. |
| StreamSpec | A validated generated-technology contract before emission. |
| Fixture | A small mod or scenario used to prove expected behavior. |
