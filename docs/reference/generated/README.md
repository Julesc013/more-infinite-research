# Generated Reference Views

Files in this directory are deterministic projections of machine-readable authorities. Run `./scripts/Update-MIRGeneratedAuthorityDocs.ps1` after changing their sources; architecture validation runs the same command with `-Check` and rejects drift.

The source authority is named at the top of each generated document. Do not edit generated tables by hand.
