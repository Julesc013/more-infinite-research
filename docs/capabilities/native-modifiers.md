# Native Modifiers Capability

Native modifier lanes observe or govern technology effects such as mining yield, laboratory productivity, robot bonuses, and logistics bonuses. These are not the same as recipe productivity for crafting machines or logistics entities.

Default posture is observe or prefer existing owners unless MIR has an explicit stream, owner policy, migration review, and fixture evidence.
