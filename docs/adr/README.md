# Architecture Decision Records

Updated: 2026-07-07

ADRs record durable 3.0 compatibility-compiler decisions. They should be short,
specific, and updated by adding a new ADR rather than rewriting history.

Initial ADR set:

- `0001`: MIR 3.0 compatibility compiler architecture.
- `0002`: Capability resolver interface.
- `0003`: DecisionRecord and StreamSpec schemas.
- `0004`: Policy overlays versus behavior scripts.
- `0005`: Generated ID and migration policy.
- `0006`: Science/lab planning policy.
- `0007`: Native modifier ownership policy.
- `0008`: Loop-risk policy.
- `0009`: Compatibility claim levels.
- `0010`: Settings and customization policy.
- `0011`: Runtime code boundaries.
- `0012`: Fixture and report-diff strategy.
- `0013`: Factorio shell and MIR compiler namespace.
