# MIR 0.10.0 Release Packet

MIR 0.10.0 is an independent finite museum reconstruction for Factorio 0.10.12 from canonical anchor `7ca81bbc98f02cfaf2d40012096d94e261e74d98`, not a feature-equivalent current MIR release and not a native-infinite parity claim. The earlier `tmp/0.10` candidate remains `SUPERSEDED-UNRELEASED` characterization evidence and is not this reconstruction's source ancestry.

## Shipped Capability

The package emits 23 finite technologies across inserter capacity, gun-turret damage, bullet shooting speed, bullet damage, and toolbelt. It uses the four target science packs. Extracted `config.lua` controls family enablement and finite level bounds.

No native infinite fields, formula counts, recipe or mining productivity, modern startup settings, dynamic discovery, or scripted approximation ship.
