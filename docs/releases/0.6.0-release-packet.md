# MIR 0.6.0 Release Packet

MIR 0.6.0 is the minimal finite museum floor for Factorio 0.6.4. Its 15 numbered technologies span inserter capacity, rocket shooting speed, and rocket damage, with loaded `config.lua` controls.

The four-file archive omits infinite fields, productivity, modern settings, dynamic discovery, and scripts. Deployment is `extract-required`.

Automated evidence passed: Factorio 0.6.4 build 5945 win64 (`8D2F7208503B88A578FE558184C87092830554FEC5AB66BCD57CBAE2F0E5A1FB`), shared negative/static gates, deterministic build, exact-ZIP extraction, and two regenerated prototype-cache proofs containing the MIR marker. Installer config was restored byte-for-byte.

Manual visual, locale-fit, and balance review is pending. Publication and tagging were not run.
