# MIR 0.12.0 Release Packet

MIR 0.12.0 is an independent finite museum reconstruction for Factorio 0.12.35 from canonical anchor `dc5bfa99249af6c9508201a4c4667063bc8e021c`. It is not feature-equivalent to current MIR and does not claim native infinite research. The earlier `tmp/0.12` candidate remains `SUPERSEDED-UNRELEASED` characterization evidence and is not this reconstruction's source ancestry.

## Shipped Capability

The package contains five levels each of inserter capacity, gun-turret damage, bullet shooting speed, and bullet damage, plus three levels each of toolbelt and logistic trash slots. It uses `science-pack-1`, `science-pack-2`, `science-pack-3`, and `alien-science-pack`. `config.lua` can disable a family or lower its emitted level count when the archive is extracted.

## Explicit Omissions

The package omits `max_level`, `count_formula`, recipe productivity, mining productivity, modern startup settings, Space Age content, dynamic discovery, and scripted approximation. It contains no development tooling, fixtures, evidence, or modern runtime source.
