# MIR 0.9.0 Release Packet

MIR 0.9.0 is an independent finite museum reconstruction for Factorio 0.9.8 from canonical anchor `6ac377389d7ffc3576fb39576dab4ace6efaec51`. Its 23 numbered technologies cover five target-proven bonus families with loaded `config.lua` controls. It does not claim native-infinite or modern parity. The earlier `tmp/0.9` candidate remains `SUPERSEDED-UNRELEASED` characterization evidence and is not this reconstruction's source ancestry.

The four-file package omits infinite fields, formula counts, productivity, modern settings, dynamic discovery, and scripts.

Automated evidence passed: exact Factorio 0.9.8 build 9400 win64 (`B76D1F8058C8EB4D3F2A0B14F5DD87356B8E1068F37F85306A834AAFA611FE23`), shared 26-case negative gate, manifest/base proof, deterministic build, ZIP-native discovery, and two independent regenerated prototype-cache proofs containing the MIR mod marker. Installer config was restored byte-for-byte.

Manual visual, locale-fit, and balance review is pending. Publication and tagging were not run.
