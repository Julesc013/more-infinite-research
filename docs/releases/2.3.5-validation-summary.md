# MIR 2.3.5 Validation Summary

The Factorio 2.0 target metadata, generated target profile, fixture metadata,
and static package gate pass. Runtime, exact archive, real-mod, differential,
and manual evidence remain pending. The candidate manifest is intentionally in
`rebuilding-after-package-visible-change` state until those gates close.
