# MIR 2.3.5 Validation Summary

## Current State

The automated Factorio 2.0 release gate is complete. The exact candidate ZIP
passed the complete target-profile suite, base and Space Age fresh-save loads,
and the real Planet Crucible/Rigor stack. Manual existing-save, settings
retention, normal-mods-directory, promotion, tag, and upload checks remain
pending.

## Differential Against 2.3.0

The immutable 2.3.0 archive contains 114 files. The 2.3.5 candidate contains
122 files: eight focused compiler, target, runtime, ownership, and graph-safety
modules were added; no shipped file was removed; 37 existing files changed.

The runtime matrix verifies retained generated IDs, effects, prerequisites,
science ingredients, settings defaults, ownership policy, package shape, and
persisted-state contracts. Expected differences are limited to the documented
3.0.5 hardening behavior, 2.3.5 metadata and public copy, and Factorio 2.0
target-profile adaptations. Factorio 2.1-only cargo streams remain absent.
