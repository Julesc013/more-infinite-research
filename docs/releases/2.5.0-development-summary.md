# MIR 2.5.0 Automatic Compiler Development Summary

MIR 2.5.0 is the semantic Factorio 2.0 companion to the accepted 3.2.0 compiler source. It begins at source commit `bae27f521070bd62a13ec54689818cd8394b3554` and preserves published MIR 2.4.0 as its immutable behavior and upgrade baseline.

The shared surface includes GenerationPlan schema 2, RecipeFactV2, relationship indexes, effect metadata, FamilyRule schema 2, CompatibilityPack schema 2, stable generated IDs, coverage accounting, command dependencies, campaign evidence, and deterministic packaging. The Factorio 2.0 target profile remains the authority for effect, stream, setting, prototype-shape, fixture, and emitter cuts.

This development record does not claim release acceptance. Required gates are clean static validation, all declared Factorio 2.0 scenarios on version 2.0.77, package determinism, the 2.4.0 to 2.5.0 save-upgrade path, selected exact modpack loads, and interactive review. No 2.1-only metadata or effect may be simulated on the 2.0 line.
