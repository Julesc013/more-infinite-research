# MIR 0.11.0 Release Packet

MIR 0.11.0 is a finite museum reconstruction for the final Factorio 0.11 release, 0.11.22. It is not feature-equivalent to current MIR and does not claim native infinite research.

## Shipped Capability

The package contains five levels each of inserter capacity, gun-turret damage, bullet shooting speed, and bullet damage, plus three toolbelt levels. It uses `science-pack-1`, `science-pack-2`, `science-pack-3`, and `alien-science-pack`. `config.lua` can disable a family or lower its emitted level count when the archive is extracted.

## Explicit Omissions

The package omits `max_level`, `count_formula`, recipe productivity, mining productivity, modern startup settings, Space Age content, dynamic discovery, and scripted approximation. It contains no development tooling, fixtures, evidence, or modern runtime source.
