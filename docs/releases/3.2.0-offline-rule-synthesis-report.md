# 3.2.0 Offline Rule Synthesis Report

MIR now has an interpretable offline rule proposal tool. The governed corpus carries explicit predicate vectors, the snapshot registry contributes unreviewed counterexamples, and the synthesis authority fixes the allowed predicates, hard predicates, objective order, production-entry gates, and non-production boundary.

The bounded search produces deterministic `REVIEW_REQUIRED` proposals for mining-drill, assembling-machine, and logistics manufacturing. Each proposal covers all positives, rejects all negatives, retains visibility, productivity eligibility, deterministic output, and loop-safety predicates, selects at least one structural family-membership predicate, and introduces no unreviewed snapshot matches.

The tool is intentionally not an applier. Its output cannot modify runtime behavior, compatibility claims, approvals, or identity state. Production entry still requires the five separately evidenced review gates.
