# MIR 0.7.0 Release Packet

MIR 0.7.0 is a finite museum reconstruction for Factorio 0.7.5. Its 23 numbered technologies span five target-proven families with loaded `config.lua` controls.

The four-file archive omits infinite fields, productivity, modern settings, dynamic discovery, and scripts. Deployment is `extract-required`.

Automated evidence passed: Factorio 0.7.5 build 7026 win64 (`EA52123915ECFB14658591876327490B094996ADE64D837929765147C4556FE8`), shared negative/static gates, deterministic build, exact-ZIP extraction, and two regenerated prototype-cache proofs containing the MIR marker. Installer config was restored byte-for-byte.

Manual visual, locale-fit, and balance review is pending. Publication and tagging were not run.
