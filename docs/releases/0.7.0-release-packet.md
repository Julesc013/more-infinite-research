# MIR 0.7.0 Release Packet

MIR 0.7.0 is an independent finite museum reconstruction for Factorio 0.7.5 from canonical anchor `7ca81bbc98f02cfaf2d40012096d94e261e74d98`. Its 23 numbered technologies span five target-proven families with loaded `config.lua` controls and do not claim native-infinite or modern parity. The earlier `tmp/0.7` candidate remains `SUPERSEDED-UNRELEASED` characterization evidence and is not this reconstruction's source ancestry.

The four-file archive omits infinite fields, productivity, modern settings, dynamic discovery, and scripts. Deployment is `extract-required`.

Automated evidence passed: Factorio 0.7.5 build 7026 win64 (`EA52123915ECFB14658591876327490B094996ADE64D837929765147C4556FE8`), the shared 26-case negative/static gates, deterministic build, exact-ZIP extraction, and two regenerated prototype-cache proofs containing the MIR marker. Installer config was restored byte-for-byte.

Manual visual, locale-fit, and balance review is pending. Publication and tagging were not run.
