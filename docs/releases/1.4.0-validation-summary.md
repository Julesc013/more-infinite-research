# MIR 1.4.0 Validation Summary

MIR 1.4.0 is the Factorio 0.14 finite-continuation and graph-safety candidate.

It is independently reconstructed from canonical anchor `dc5bfa99249af6c9508201a4c4667063bc8e021c`; it does not inherit the Factorio 0.15 source tree and does not claim native-infinite parity.

Unsupported recipe-productivity settings are omitted rather than displayed as inert controls. The archive is byte-reproducible and sealed to the finite-reconstruction profile, harness, and exact Factorio binary.
