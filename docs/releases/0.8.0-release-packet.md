# MIR 0.8.0 Release Packet

MIR 0.8.0 is a finite museum reconstruction for Factorio 0.8.8. It ships 23 numbered technologies across five target-proven families with loaded `config.lua` controls.

The four-file archive omits infinite fields, formula counts, productivity, modern settings, dynamic discovery, and scripts. Deployment is `extract-required`; the target does not discover the ZIP directly.

Automated evidence passed: Factorio 0.8.8 build 8138 win64 (`A4226859DA746314A59D49937444F37A64828A7ABEC46AB3FC05E836C3F4C096`), shared negative/static gates, deterministic build, exact-ZIP extraction, and two regenerated prototype-cache proofs containing the MIR marker. Installer config was restored byte-for-byte.

Manual visual, locale-fit, and balance review is pending. Publication and tagging were not run.
